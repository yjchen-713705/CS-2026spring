# ALU_Seq 项目说明

## 项目结构

| 文件名 | 功能描述 |
|--------|----------|
| ALU_Seq.v | 主模块文件，实现8种基本运算功能 |
| ALU_Seq_tb.v | 测试文件，包含10个测试用例验证ALU功能 |
| README.md | 项目说明文档 |

## 文件内容说明

### 1. ALU_Seq.v

- **模块名称**：ALU_Seq
- **功能**：实现8种基本运算，在时钟上升沿完成操作
- **输入端口**：
  - clk：时钟信号，上升沿触发
  - a：32位操作数A
  - b：32位操作数B
  - op：3位操作码
- **输出端口**：
  - result：32位运算结果
  - cout：进位/溢出标志
- **支持的运算**：
  - 000：加法
  - 001：减法
  - 010：乘法
  - 011：与运算
  - 100：或运算
  - 101：异或运算
  - 110：非运算
  - 111：右移运算

### 2. ALU_Seq_tb.v

- **功能**：测试ALU_Seq模块的功能
- **测试用例**：
  1. 5 + 3 = 8，cout=0
  2. FFFFFFFF + 1 = 0，cout=1
  3. 5 - 3 = 2，cout=0
  4. 3 - 5 = FFFFFFFE，cout=1
  5. 5 * 3 = 15，cout=0
  6. 5 & 3 = 1，cout=0
  7. 5 | 3 = 7，cout=0
  8. 5 ^ 3 = 6，cout=0
  9. ~5 = FFFFFFFA，cout=0
  10. 5 >> 1 = 2，cout=0
- **仿真设置**：时钟周期为10ns

## 使用方法

1. 在Vivado中创建新项目
2. 添加ALU_Seq.v和ALU_Seq_tb.v文件
3. 运行仿真验证功能
4. 查看仿真波形和输出结果
